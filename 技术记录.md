## 使用ShadowsocksR提升Github下载速度

1. 开启sock5服务



   ![01](E:\github\anjiel\images\01.jpg)

2. 配置git代理

   ```
   git config --global http.proxy 'socks5://127.0.0.1:1080' 
   git config --global https.proxy 'socks5://127.0.0.1:1080'
   git config --global -l
   ```

3. 重新clone即可
